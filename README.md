Envoy Java Client
========


Publish
------------
This project needs to be published to artifactory.

``` bash
$ mvn deploy
```

Import
------------
Maven

``` XML
<dependency>
    <groupId>com.sazze.en</groupId>
    <artifactId>envoy-client-java</artifactId>
    <version>VERSION</version>
</dependency>
```

sbt

``` Scala
"com.sazze.en" % "envoy-client-java" % "VERSION"
```
